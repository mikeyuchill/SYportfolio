Project Overview

LionMatch is a microservices-based dating web app designed exclusively for Columbia University students. The platform aims to create a safe and engaging matchmaking experience tailored to the university community, leveraging modern full-stack technologies for seamless interactions and scalability.

LionMatch provides an innovative dating experience tailored to university students, ensuring a secure, engaging, and data-driven matchmaking process. By leveraging microservices architecture, cloud deployment, and modern frontend/backend technologies, this project serves as a strong demonstration of full-stack engineering capabilities.

My Contributions to LionMatch

As a full-stack developer and project lead, I contributed to LionMatch in the following ways:

• System Architecture & Backend Development:

  • • Designed a microservices-based architecture to ensure scalability and modularity.
  • • Developed backend RESTful APIs using Flask, enabling seamless data flow between the frontend and database.

• Frontend Development & UI/UX Design:

  • • Coded the frontend baseline using React, ensuring a dynamic and user-friendly experience.
  • • Designed intuitive UI/UX components to enhance user interaction and future scalability.

• Cloud & Database Management:

  • • Deployed the application on AWS, utilizing services like EC2, S3, and Lambda to ensure reliability and performance.
  • • Integrated a PostgreSQL/Firebase database for secure and efficient data storage.

• Project Management & Leadership:

  • • Acted as the pivot point, coordinating team efforts and managing project timelines.
  • • Successfully delivered the project one week ahead of schedule through effective task delegation and communication.
